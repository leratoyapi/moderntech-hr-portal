# ModernTech HR Portal

> Module 1 Core Project | YouthCode | Life Choices Academy | Group 14

---

## About The Project

ModernTech Solutions is a software development company that has grown from 20 to 250 employees. Their HR processes – currently managed through spreadsheets, emails, and Word documents – have become fragmented and inefficient.

This project addresses their pain points by delivering a **front-end web-based HR system** that centralises employee data, automates payroll, manages time-off requests, and tracks attendance.

## Current Pain Points

-  Employee data is scattered across multiple Excel files
- Time-off requests managed via emails and shared calendars
- Payroll calculated using complex interconnected spreadsheets
- Performance reviews stored as Word files in a shared drive

---

## Project Goal

To create a **user-friendly, responsive web-based HR system** that demonstrates how ModernTech's HR processes can be centralised and automated using modern front-end technologies.
